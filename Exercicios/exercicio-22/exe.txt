1. Dê uma definição para o elemento HTML <body>.
R: O <body> é o corpo que contém toda a estrutura da página html.

2. É correto dizer que por padrão o todos os elementos HTML possuem uma definição de margin e padding?
R: Não. Nem todos os elementos possuem.

3. Uma tag <figure> pode conter várias tags <figurecaption>?
R: Não. O correto é utilizar uma tag <figurecaption> para cada elemento <figure>.
